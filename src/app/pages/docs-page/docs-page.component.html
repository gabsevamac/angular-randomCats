General information

This API is a non-commercial project developed as part of a college assignment. Its primary purpose is to serve as a practical implementation for learning and showcasing concepts related to API design, development, and documentation. The API is designed exclusively for academic use, aiming to demonstrate functionality and best practices in building RESTful services.
This project does not involve any commercial intent or real-world application beyond the educational scope. Its features and endpoints are tailored to meet the requirements of the assignment and to provide a hands-on experience in working with modern web development technologies.
